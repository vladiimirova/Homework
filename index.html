Види селеторів

  • p { color: black; } - стиль для всіх параграфів на сторінці
  • #myElement { font-size: 16px; } - стиль для елементу з конкретним id
  • .myClass { background-color: yellow; } - стиль для всіх елементів з конкретним класом
  • .class1.class2 { border: 1px solid red; } - стиль для елементів, які мають обидва класи
  • div p { margin: 10px; } - стиль для елементів, які знаходяться всередині іншого елемента
